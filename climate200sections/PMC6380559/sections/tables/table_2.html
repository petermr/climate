Vegetation metrics refer to change in the metric between surveys. Explanatory variables refer to the mowing treatment (unmown or mown), and the interaction between mowing treatment and time (years) between surveys. T values are shown for each variable in each model.

Significance of the variables is shown by asterisks:

* = P<0.05;

** = P<0.01;

*** = P<0.001.

Df = 59 for all models.
